Testing EA-Equivalence
----------------------

An implementation of Kaleyski's algorithm for testing EA-equivalence between two functions
is available in the repository:

- ``github/heggebakk/ea-equivalence``

Testing Linear and Affine Equivalences
--------------------------------------

An adapted implementation of Kaleyski's approach for linear and affine equivalence testing,
including EA-equivalence tests for quadratic APN functions via orthoderivatives, is available at:

- ``github/heggebakk/affine-equivalence``
