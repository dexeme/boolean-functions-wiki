Background and definition[edit | edit source]

Almost perfect nonlinear (APN) functions are the class of (𝑛,𝑛) Vectorial Boolean Functions that provide optimum resistance to against differential attack. Intuitively, the differential attack against a given cipher incorporating a vectorial Boolean function 𝐹 is efficient when fixing some difference 𝛿 and computing the output of 𝐹 for all pairs of inputs (𝑥₁,𝑥₂) whose difference is 𝛿 produces output pairs with a difference distribution that is far away from uniform.

The formal definition of an APN function 𝐹 : 𝔽2𝑛 → 𝔽2𝑛 is usually given through the values

Δ F ( a , b ) = | { x ∈ F 2 n : F ( x ) + F ( a + x ) = b } | {\displaystyle \Delta _{F}(a,b)=|\{x\in \mathbb {F} _{2^{n}}:F(x)+F(a+x)=b\}|}

which, for 𝑎≠0, express the number of input pairs with difference 𝑎 that map to a given 𝑏. The existence of a pair ( a , b ) ∈ F 2 n ∗ × F 2 n {\displaystyle (a,b)\in \mathbb {F} _{2^{n}}^{*}\times \mathbb {F} _{2^{n}}} with a high value of Δ𝐹(𝑎,𝑏) makes the function 𝐹 vulnerable to differential cryptanalysis. This motivates the definition of differential uniformity as

Δ F = max { Δ F ( a , b ) : a ∈ F 2 n ∗ , b ∈ F 2 n } {\displaystyle \Delta _{F}=\max\{\Delta _{F}(a,b):a\in \mathbb {F} _{2^{n}}^{*},b\in \mathbb {F} _{2^{n}}\}}

which clearly satisfies Δ𝐹 ≥ 2 for any function 𝐹. The functions meeting this lower bound are called almost perfect nonlinear (APN).

Walsh transform[1][edit | edit source]

Any (𝑛,𝑚)-function 𝐹 satisfies

∑ a ∈ F 2 n , b ∈ F 2 m ∗ W F 4 ( a , b ) ≥ 2 2 n ( 3 ⋅ 2 n + m − 2 m + 1 − 2 2 n ) {\displaystyle \sum _{a\in \mathbb {F} _{2^{n}},b\in \mathbb {F} _{2^{m}}^{*}}W_{F}^{4}(a,b)\geq 2^{2n}(3\cdot 2^{n+m}-2^{m+1}-2^{2n})}
