Testing EA-equivalence[1]

Implementation of Kaleyskis algorithm testing ea-equivalences between two functions can be found at github/heggebakk/ea-equivalence

Testing equivalences [2]

An adapted implementation of Kaleyskis algorithm for testing for Linear and Affine equivalences, alongside with testing two quadratic APN functions for EA-equivalence through their orthoderivatives can be found at github/heggebakk/affine-equivalence

Notes

  1. ↑ Kaleyski. (2021). Deciding EA-equivalence via invariants. Cryptography and Communications, 14(2), 271–290. https://doi.org/10.1007/s12095-021-00513-y
  2. ↑ Heggebakk. (2022). An efficient implementation of a test for EA-equivalence, 43-49. The University of Bergen. https://hdl.handle.net/11250/3003709
